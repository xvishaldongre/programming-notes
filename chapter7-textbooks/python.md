## preliminary reading

* [Learn Python: the Hard Way](http://learnpythonthehardway.org/book/) - A free Python learning tutorial published by a foreign programmer. The author advocates learning from practice, and the style of traditional programming books is very different. Suitable as an introductory tutorial for Python.

* [A Byte of Python] (http://www.swaroopch.com/notes/python/) - Ibid., also a book that is very suitable for getting started. It is more extensive than the previous book. The author is Indian. The writing style is similar to the traditional compilation.

* [Zero Basics Python] (https://github.com/qiwsir/ITArticles/blob/master/BasicPython/index.md) - The old Python introductory tutorial, some users put it on Gitbooks, poke [ Zero-based Python] (http://looly.gitbooks.io/python-basic/)

* [Learning Python] (http://book.douban.com/subject/22139956/) - Compared to the previous books, this book has a certain depth. 
From the thickness of the book, the author is not Pythonic at all, 1000+ is too thick and too thick for Python entry books! 
! 
In fact, the author does not simply introduce how to use Python in the book, but also talks about the design ideas behind it and some best practices. For beginners of programming, such experience is very applicable.

## Related Websites

* [vinta/awesome-python](https://github.com/vinta/awesome-python) - If you want to illustrate the power of Python from the side, take a look at this.
